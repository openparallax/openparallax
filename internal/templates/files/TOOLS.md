# Tools

Available capabilities and their descriptions.

## File Management
Read, write, delete, move, copy, create directories, list, and search files.

## Shell Execution
Run shell commands with Shield evaluation. Timeout: 30 seconds.

## Web Research
HTTP requests with content extraction.

## Browser Automation
Navigate, click, type, extract content, and take screenshots.

## Notes and Memory
Save, recall, and search accumulated knowledge.

## Calendar
Read, create, update, and delete calendar events (Google Calendar + CalDAV).

## Git Operations
Status, diff, commit, push, pull, log, branch, and checkout using pure-Go git.

## Text Processing
Summarize, extract, transform, and translate text.

## Canvas
Generate files: Mermaid diagrams, SVG, Markdown, HTML.

## Scheduled Tasks
Create, list, and delete recurring tasks via HEARTBEAT.

## Email
Compose and send emails via SMTP.
