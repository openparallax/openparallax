# @openparallax/memory

Semantic memory with FTS5 search and vector embeddings

TypeScript wrapper for the [OpenParallax](https://docs.openparallax.dev) memory module. Communicates with a pre-built Go binary over JSON-RPC (stdin/stdout).

## Installation

```bash
npm install @openparallax/memory
```

## Documentation

See the [memory documentation](https://docs.openparallax.dev/memory/) for the full API reference and examples.
