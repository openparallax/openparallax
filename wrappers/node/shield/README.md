# @openparallax/shield

4-tier AI security pipeline (policy, classifier, LLM evaluator, human approval)

TypeScript wrapper for the [OpenParallax](https://docs.openparallax.dev) shield module. Communicates with a pre-built Go binary over JSON-RPC (stdin/stdout).

## Installation

```bash
npm install @openparallax/shield
```

## Documentation

See the [shield documentation](https://docs.openparallax.dev/shield/) for the full API reference and examples.
